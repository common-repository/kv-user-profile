== Description ==

This is a simple Plugin to add custom User Photo or Profile Picture to any users. This plugin helps you to add profile picture while creating,editing users from the backend. 

Simple Custom User Photo Plugin helps you to make changes from the add new users page. It override,if you have default Gravatar on your site.If you have any further inquires and feebacks, Please contact me here at <a href="http://www.kvcodes.com/contact-me"> Kvcodes</a> 


= Features =
* Easy use
* Easy to add User Photos
* No Setup, Just Activate and Play it on <b>Add New User</b> Page.
* Works with Remote Image, You can just use the URL of the Direct Remote image.

 
== Installation ==

1. Upload `kv-user-profile` to the `/wp-content/plugins/`  directory
2. Activate the Plugin through the 'Plugins' menu in WordPress
